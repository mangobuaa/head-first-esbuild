# Content Types
下面列举了 esbuild 所有的内容类型。每一种类型都对应一个 loader，esbuild 使用 loader 加载对应的内容。
默认情况下，esbuild 可以根据文件的扩展名获取 loader。
loader 也可以不使用默认值，通过配置文件指定。
## JavaScript
> loader: `js`



## TypeScript
> loader: `ts` or `tsx`



## JSX
> loader: `jsx` or `tsx`


## JSON
> loader: `json`


## CSS
> loader: `css`

## TEXT
> loader: `text`


## Binary
> loader: `binary`

## Base64
> loader: `base64`

## Data URL
> loader: `dataurl`

## External file
> loader: `file`
